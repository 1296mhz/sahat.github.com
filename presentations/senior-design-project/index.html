AudioStream

Synchronized audio playback on multiple devices

by Sahat Yalkabov

Objective

AudioStream lets you upload any MP3 song and playback it across multiple client devices. (e.g. laptops, smartphones, desktops)

Oh hey, these are some notes. They'll be hidden in your presentation, but you can see them if you open the speaker notes window (hit 's' on your keyboard).

Use Case

  • You have decided to hack on a project with 4 friends.
  • Each person has music on their laptop.
  • One of them can create an AudioStream virtual room and upload some songs.
  • Four other members could then join in, and have music playing simultaneously on all 5 laptops in a synchronized manner.

Tools

  • Node.js - back-end
  • Backbone.js - front-end
  • MongoDB - database storage
  • HTML5 WebSockets - bidirectional server-client communication
  • WebAudio API - real-time audio processing
  • Dropbox API - for storing music files
