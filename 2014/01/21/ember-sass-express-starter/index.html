Ember+Sass+Express Starter

The music band website

Source Code

This project provides a starting point for your Ember apps with Express web framework serving as a RESTful API back-end. Directory structure is heavily influenced by ember-tools and Ember App Kit. In fact it is designed to work with ember-tools for quickly scaffolding models, views, template, routes and controllers.

Features:

  • Token-based authentication
  • Sample REST API
  • Neat application structure
  • Bootstrap

Technology Stack: Ember.js, Node.js, Sass, MongoDB
